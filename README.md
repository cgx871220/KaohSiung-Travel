# kaohsiung-travel
利用 Vite 建立專案、使用 AJAX 撈取 API 資料、Bootstrap 格線系統排版



## 使用技術

* HTML/CSS
* Scss
* Bootstrap
* Javascript
* Vue


## 內容說明
在這個專案中，我運用了 Vite 作為建立前端專案的工具，同時透過 AJAX 技術從外部 API 撈取資料,並使用了 Bootstrap 的格線系統來實現簡單且響應式的網頁排版
